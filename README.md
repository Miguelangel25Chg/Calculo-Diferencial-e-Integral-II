## 🧠 Fórmulas Fundamentales

Para que no haya pierde con las imágenes, aquí tienes el respaldo técnico en Markdown (LaTeX):

### 1. Linealidad de la integral
$$\int [f(x) \pm g(x)] dx = \int f(x) dx \pm \int g(x) dx$$
$$\int k \cdot f(x) dx = k \int f(x) dx, \quad k \in \mathbb{R}$$

### 2. Integrales inmediatas
* **Potencia:** $\int x^n dx = \frac{x^{n+1}}{n+1} + C, \quad n \neq -1$
* **Logarítmica:** $\int \frac{1}{x} dx = \ln|x| + C$
* **Exponencial:** $\int e^x dx = e^x + C$
* **Trigonométricas:**
    * $\int \sin(x) dx = -\cos(x) + C$
    * $\int \cos(x) dx = \sin(x) + C$
    * $\int \sec^2(x)  dx = \tan(x) + C$

### 3. Métodos básicos de integración
* **Integración por partes:** $\int f g' = fg - \int g f'$
* **Cambio de variable:** $\int f(g(x))g'(x) dx = \int f(u) du, \quad u = g(x)$
